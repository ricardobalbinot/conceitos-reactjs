## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/ricardobalbinot/conceitos-reactjs.git

# Go into the repository
$ cd conceitos-reactjs

# Install dependencies
$ yarn

# Run site at port 8080
$ yarn start

# Run CRUD tests
$ yarn test
```
